# evaluation_example_vpibench

This directory contains our **OSWorld-based implementation** of the VPI-Bench setup, aimed at fair comparison in a unified environment and supporting community follow-up research.

Current baseline task set in this repository:

- `examples/computer_use_osworld/*.json`: task configs used by this baseline
- `test_all.json`: task index used by this repository
- `test_all_computer_use_osworld.json`: original index from the source split

References:

- OSWorld: https://github.com/xlang-ai/OSWorld
- VPI-Bench: Visual Prompt Injection Attacks for Computer-Use Agents: https://arxiv.org/abs/2506.02456
